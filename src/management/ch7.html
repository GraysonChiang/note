# 第七章 管理變革與創新

# 組織推動變革的影響力量

  • 內部力量
  • 外部力量

# 組織變革的類型

  • 結構變革：報告對象、合作機制、員工授權與職位重劃分
  • 技術變革：工作程序、方法與設備的修改
  • 人員變革：員工個人或團體之態度、期望、認知及行為的改變

# 組織進行變革時，通常組織成員抗拒變革的理由為何？

  • 不確定性
  • 慣性
  • 損及個人利益
  • 認為變革對組織沒有好處
